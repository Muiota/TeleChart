Tele chart
====================
Demo: https://muiota.github.io/TeleChart/

### Features:
 - Pure js (without css)
 - Small size (<10kB)
 - Advanced zoom
 - Advanced navigation
 - Vertical auto zoom
 - Scale axes
 - Day/night switch
 - High resolution (fix blur, retina support)
 - Switch series
 - Smooth animation
 - Smart X-axis labels
 - Mobile high resolution & touches support
 - Material design animations
